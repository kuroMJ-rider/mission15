## 🔧 데이터 전처리

### 결측치 처리 — 맥락 기반 처리

기계적 중앙값 대체 대신, 결측 고객의 특성을 먼저 파악 후 케이스별 처리

| 컬럼 | 결측 수 | 처리 방식 | 근거 |
|---|---|---|---|
| CREDIT_LIMIT | 1건 | 행 삭제 | 1건이므로 삭제가 가장 안전 |
| MINIMUM_PAYMENTS (PAYMENTS=0) | 240건 | 0으로 채움 | 결제 이력 없으면 최소결제도 0 |
| MINIMUM_PAYMENTS (PAYMENTS>0) | 73건 | 행 삭제 | 결제했는데 최소결제액 없음 → 신뢰 불가 |

> 중앙값으로 기계적 대체 시, 소비 패턴이 다른 고객군이 섞여 클러스터링 품질 저하 우려

### 이상치 처리 — 유지 결정

PURCHASES max 49,039 / CASH_ADVANCE max 47,137 등 극단값 존재.
**처리 방향: 유지** — 극단값 고객은 노이즈가 아닌 핵심 군집(VVIP, 현금서비스 의존형)의 구성원.
StandardScaler 적용으로 스케일 영향 완화.

## 🤖 모델링

### 알고리즘 비교

Elbow Method(k=2~15) + Silhouette Score(k=2~9) 탐색 → k=4, k=5 프로파일 비교 → **k=5 확정**

| 알고리즘 | 실루엣 스코어 | 비고 |
|---|---|---|
| K-means 원본 (k=5) | 0.1932 | 기준 모델 |
| 계층적 클러스터링 (k=5) | 0.1361 | ward linkage, 500개 샘플 덴드로그램 |
| **PCA K-means (k=5)** | **0.2604** | **✅ 최종 선정** |
| DBSCAN | — | 클러스터 쏠림 심함 → 제외 |
| GMM | — | 대부분 1.0 대 0.0으로 이분화 → 제외 |

### PCA 설정

| 용도 | 주성분 수 | 설명력 |
|---|---|---|
| 시각화용 | 2개 | 47.6% |
| 클러스터링용 | **6개** | **76.0%** |

효과: 다중공선성 컬럼 압축 → 노이즈 제거 → 실루엣 스코어 향상

## 💡 핵심 인사이트

### 파레토 구조 발견

> **8.7%의 고객(VVIP + 일반 활성형)이 전체 구매액의 48.1%를 차지한다.**

한스 로슬링(팩트풀니스)의 80/20 법칙이 이 데이터에서도 작동 — 마케팅 예산은 이 8.7%를 먼저 처리해야 한다.

### 가설 1 — 구매 방식이 소비력을 가른다 ✅ 검증됨

| 구매 방식 | 비율 | PURCHASES 평균 | 특징 |
|---|---|---|---|
| 혼합 | 31% | 2,271 | 최상위 소비 — 카드 극대화 활용 |
| 일시불 | 21% | 789 | 중상위 — 즉시 결제 선호 |
| 현금전용 | 23% | 0 | 취약 — 카드를 부채 수단으로 사용 |
| 할부 | 25% | 542 | 건전 저소비 — 전액결제 비율 최고(0.26) |

**혼합 구매 여부가 고소비 고객을 가르는 핵심 변수**임이 확인됨

### 가설 2 — VVIP 승급 후보군 발굴 ✅ 부분 검증됨

- VVIP 정의: 혼합 구매 + PURCHASES 상위 0.2% + CASH_ADVANCE 중앙값 = 0
- **승급 후보군 70명** 발굴 (혼합 구매 + PURCHASES/CREDIT_LIMIT/PAYMENTS 상위 10% + CASH_ADVANCE = 0)
- 후보군 CREDIT_LIMIT = VVIP의 94% 수준 → 한도는 충분, **소비 규모 확대**가 승급 핵심 조건
